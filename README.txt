Sistema para controle de estoque, Armazena e gera dados de Produtos, Clientes e fornecedores.

#Features
-Desenvolvido em PHP, HTML. Com uma fácil manutenção e adesão de novas funcionalidades.
-Gera relatório de produtos, clientes e fornecedores para a mais fácil visualização de dados.
-Futuras implementações mais avançadas sobre planos de negocios, como, produtos mais vendidos, clientes que compram mais.
-Open Source, 100% Grátis.
-Ajude no desenvolvimento, para que façamos uma comunidade open source mais forte.

#Como executar
-Tenha um servidor MySQL e apache, recomendo o uso do Xampp.
-Faça o download dos arquivos.
-Coloque em alguma pasta especifica para a execução, no caso do xampp o "C:\xampp\htdocs".
-Inicie os servidores apache e MySQL.
-Inicie uma conexão local ou externa.
-Acesse o site e realize suas operações.
